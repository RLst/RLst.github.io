BhaVE

Behaviour Tree Visual Editor

BhaVE is my behaviour editor that I wrote and designed from the ground up. I was tasked with doing the AI in my current major group project and I decided to create an editor for it.

The entire system is highly flexible and designed to be generic so that it can be virtually adapted to any Unity project. This is my premiere project that I worked extremely hard on and am very proud of, and I plan to someday release it on the Unity Asset Store.
